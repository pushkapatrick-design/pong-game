# Pong + Quiz Game

A single-player Pong game with quiz questions integrated.

## How to host on GitHub Pages

1. Create a new **public repository** on GitHub (e.g., `pong-quiz`).
2. Upload these two files (`index.html` and `README.md`) directly to the repo.
3. Go to **Settings → Pages**.
4. Under **Source**, select the **main branch** and **/(root)** folder, then click Save.
5. After 1–2 minutes, your game will be live at:
   `https://yourusername.github.io/pong-quiz/`

Enjoy!
